# Introduction

Welcome to Authy-JS! A modular authentication library supporting multiple frameworks and databases.

## What is Authy-JS?

This library supports multiple frameworks and databases for storing sessions, enabling easy integration and customization for various use cases.

#### Key Features

-   Multi-framework support (Express, Fastify, Next.js, etc.)
-   Flexible adapters for different session databases (Redis, MongoDB, etc.)
-   Easy-to-use handlers for integrating with various frameworks
-   Easily add support for other databases and frameworks.
